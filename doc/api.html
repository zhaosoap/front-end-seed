/run/clean post

Execution of available clean in TelData-LocVis.

post

execute a clean in TelData-LocVis.

post /run/clean

execute a clean in TelData-LocVis.

  • Request
  • Response

Query Parameters

  • access_token: required (string)

    The access token provided by the authentication application

    Example:

    AABBCCDD

Body

Type: application/json

Schema:

{
  "type": "object",
  "$schema": "http://json-schema.org/draft-03/schema",
  "id": "http://jsonschema.net",
  "required": true,
  "properties": {
    "id": {
      "type": "string",
      "required": true,
      "minLength": 36,
      "maxLength": 36
    },
    "input":{
      "type": "string",
      "required": true,
      "minLength": 36,
      "maxLength": 36
    },
    "configurations": {
      "type": "array",
      "required": true,
      "items": {
        "type": "object"
      }
    }
  }
}

Example:

{
  "id": "clean",
  "input": "forword.csv",
  "configurations": [
    {"notNull1": "true"},
    {"notNull2": "False"}
  ]
}

HTTP status code 200

Body

Type: application/json

Example:

{ 
  "message": "The clean has been properly executed",
  "result": "out123.csv"
}
